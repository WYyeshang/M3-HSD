Our codebase is built upon the Time-Series-Library. Please follow their instructions to set up the basic environment.

Clone the repo:

Bash
git clone https://github.com/your_name/M3-HSD.git
cd M3-HSD
Install dependencies:
You can refer to the environment requirements in thuml/Time-Series-Library. Specifically, we recommend:

Python 3.8+

PyTorch 1.12.1+

CUDA 11.3+

Other common libs: matplotlib, numpy, pandas, scikit-learn.

Bash
pip install -r requirements.txt
